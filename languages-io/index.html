各语言输入输出注意事项

2017-07-24

做题的基础，记录一些输入输出的方法。

个人学习总结，只会一些 C/C++、Java、Python。目的是完成 OJ 上的题目，所以并不详细。

C

普通输入输出使用C++中的 cin 与 cout 即可，但控制输出位数时使用 printf 比较方便，要记得 #include <cstdio>

printf

  • %ld 输出长整型；%Lf输出长双精度浮点型
  • 使用 %% 来打印一个 %
  • float 与 double 类型都以 %f 指定输出(printf会把float自动转化为double)
  • %f 最多输出小数点后6位
  • %m.nf 此处m, n皆为整数。
    m 指定最小位宽，小数点也占一位。数据位宽大于m时，按实际位宽输出；数据位宽小于m时，m为正数则右对齐左补空格，m为负数则左对齐右补空格。
    n 控制小数位数

scanf

  • scanf("输入格式控制字符串", 输入地址表)
  • %f 输入 float; %lf 输入 double
  • 可指定截取位宽但不能指定小数位数
  • %*2d表示跳过2个int
  • 普通字符原样输入
    • "%d %d" 表示以空格/Tab/回车作为输入数据的分隔符输入
    • "%d,%d" 表示以逗号作为数据分隔符输入，其他普通字符同理
  • 由于 scanf() 把空格/Tab/回车等作为分隔符，读到这些字符时系统会认为读入结束，因此不能读入带空格的字符串
  • 以任意分隔符输入算式 scanf("%d %c%d", &data1, &op, &data2);

putchar 与 getchar

  • putchar(ch)向屏幕输出一个字符
  • ch = getchar() 从键盘接收一个字符保存在变量ch中
  • 一些情况下使用 getchar(); 来吸收输入缓冲区中的回车字符，避免被后面的变量当作有效字符读入。(输入过程：键盘->输入缓冲区–按回车–>内存) 另一种解决方案为把下一条语句改成 scanf(" %c", &ch); 即 %c 前加一个空格

gets 与 puts

  • puts(str); 与 printf("%s", str); 等价
  • gets(str); 可以输入带空格的字符串，因其把空格和制表符当成字符串的一部分。把回车作为字符串的终止符，同时将回车从缓冲区读走但不作为字符串的一部分
